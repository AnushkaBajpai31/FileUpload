Steps to create File Upload Project (using Angular 2+ and Flask)

1. Create a new project in Angular using 'ng new project-name'.

2. Create a basic HTML file with an 'input' tag of type 'file' in 'form' tag.

3. Create a method to store all files in a list after choosing files.

4. Create a new project in Flask.

5. Start by creating 'app.py'.

6. Create an endpoint of upload API which will upload files.

7. Create a service in Angular project.

8. Add function to call upload API.

9. Create a method to upload files on submit.

10. Call the upload API method of the upload service while calling submit method.
